Welcome to BASICSS

Welcome to BASICSS, a simple little CSS framework built using 100% vanilla CSS. The framework is very lightweight, and is built with CSS variables offering extensive, easily editable properties.

Setup Instructions:

  1. Include the font link in the head of your HTML file. 

                            
<link href='https://fonts.googleapis.com/css?family=Open+Sans:400,300,300italic,400italic,600
,600italic,700,700italic,800,800italic
&subset=latin,latin-ext' rel='stylesheet' type='text/css'>

  2. Include the main.css file within the head of the HTML document.
